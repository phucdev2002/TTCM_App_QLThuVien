                                                                  PHẦN MỀM QUẢN LÝ THƯ VIỆN

Hướng dẫn cài đặt và build chương trình:
1.	Cài đặt môi trường 
-	Download Visual Studio 2022 tại: https://visualstudio.microsoft.com/ sau đó cài đặt trực tiếp trên máy.
-	Download .NET 6 SDK tại: https://dotnet.microsoft.com/en-us/download/dotnet/6.0 
-	Test thử đã cài đặt thành công hay chưa, bằng cách mở cửa sổ CMD gỗ lệnh: dotnet –list-sdks
-	Download thư viện DevExpress tại: https://www.devexpress.com/ 
-	Cài DevExpress theo video sau: https://www.youtube.com/watch?v=bPZp6FB6pn8&t 

2.	Build source code
-	Mở file database “QLThuVien” cài dữ liệu vào SQL server 
-	Mở file QuanLyThuVien.sln, tìm đến thư mục DAO, mở file DataProvider.cs
-	Thay đổi đường dẫn liên kết cơ sở dữ liệu của SQL của máy
-	Sau đó ta Run chương trình
